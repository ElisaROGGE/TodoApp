# ✅ Todo List App — Java Swing

A clean, dark-themed desktop task manager built with Java Swing.  
This is my first Java portfolio project, built from scratch while learning the language.

## Features

- **Add tasks** by typing and pressing Enter or clicking the button
- **Mark tasks as done** — completed tasks are highlighted in green
- **Delete tasks** with a confirmation dialog
- Dark theme UI with a custom color palette

---

## Getting Started

### Prerequisites

- [Java JDK 17+](https://adoptium.net) installed on your machine

Verify your installation by running:
```bash
java -version
```

### Installation

1. Clone the repository
```bash
git clone https://github.com/ElisaROGGE/TodoApp
```

2. Run the application

** VS Code**
Open `TodoApp.java` and click the ▶ **Run Java** button in the top right corner.
